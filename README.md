# My Restaurant 

A fully responsive and dynamic restaurant website built with **HTML5, Tailwind CSS, and Vanilla JavaScript**, featuring **API integration, pagination, search, filters, and a shopping cart**.

## 🌐 Live Demo
🔗 *[Preview Link](https://ahmedtika74.github.io/my-restaurant/)*

## 📌 Project Overview
This project demonstrates the ability to build a **fully interactive restaurant web application**. Users can:  

- Browse menu items fetched from an **API**  
- Use **pagination** to navigate through menu items  
- **Search** for specific dishes  
- **Filter menu items by category**  
- **Add items to a cart** and manage their selection  

The project shows how a **UI design is converted into a dynamic, responsive website** with modern frontend practices.

## 🛠 Technologies Used
- **HTML5** – Semantic markup and structure  
- **Tailwind CSS** – Styling, layout, and responsive design  
- **JavaScript (Vanilla)** – API integration, DOM manipulation, pagination, search, filters, and cart functionality  
- **Responsive Design** – Mobile-first approach

1. Clone the repository:
```bash
git clone https://github.com/ahmedtika74/my-restaurant.git
```
2. Open index.html in your browser
3. Ensure the API endpoint in script.js is accessible
